Распиши как создать классы для рисования 2D - графиков. Основной класс графика, наследуются от sf::Drawable и sf::Transformable. Также необходимо создать класс сетки, осей, кривых. У графика есть область рисования, она обведена черным контуром. Над этой областью по середине выводится наименование графика, под областью рисования расположены подписи оси X для сетки, слева расположены по оси Y для сетки, правее подписей вертикально расположено название оси Y, для оси X название оси расположено под подписями. Как выбирается масштаб? По умолчанию не рисуется сетка и подписи, так как нет данных. При наличии данных, вычисляются максимальное и минимальное значения. По ним определяется масштаб по оси Y. Длина данных указывает масштаб графика по оси X. Должна быть возможность отключить отображение осей, сетки, названия графика. при отключении необходимо перерисовать график, изменив размер с учетом отключенных элементов графика. Распиши максимально подробно каждый класс (Plot, Grid, Axis, Curves).
